Spotlight Rules

An openly-governed, well-maintained build of the Spectral API linter — a maintained CLI, a standalone ruleset specification, a VS Code extension, and a browser governance playground, under API Commons.

spotlight-spec

Specification

A standalone specification of the ruleset format — rules and rulesets — with a JSON Schema for validation, and an explorable catalog of every rule by artifact.

CLI & engine

The linter for JSON/YAML API descriptions — OpenAPI, AsyncAPI, and Arazzo — with custom rulesets. A maintained fork of Spectral, published to npm.
